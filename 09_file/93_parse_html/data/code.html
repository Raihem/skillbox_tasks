Видеолекции

Смотреть видеоуроки можно в любое время, доступ к учебным материалам предоставляется бессрочно.

Практика

Новый материал закрепляется с помощью практического задания, которое будет проверено за 48 часов.

Консультации

Живое общение с преподавателями и другими студентами в мессенджерах для обсуждения заданий и обмена опытом.

Диплом

Итоговая работа по заданию от реальных заказчиков позволяет применить свои навыки для решения существующих бизнес-задач.
